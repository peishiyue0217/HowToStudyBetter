# **HowToStudyBetter**
# 中国学生（高中）内卷指南<br>

---

- #### 一、 我为什么要内卷？<br><br>
 
    - 1. 我为什么要学习？<br><br>
 
         - 1. *提升人生境界*——学习可以让我们的眼界更宽广，接触更多高质量的圈子。比如小时候玩的很好的同学，你有没有想过再见面会是什么场景？如果差别很大的话，那就是所在圈子的差别造成，怎么弥补这个差别呢？坚持学习，提高自己的人生境界，提高自己的眼界，进入更高的学府，取得进入牛人圈的“通行证”，这样才可以有更美好的成功。 <br><br>
 
         - 2. *体会人生快乐*——学习在上一辈子是渴望却很少人做到的事情，而现在学习是多么简单的事情，随处可以学习，现在互联网这么发达，很多想学的知识也是随时可以学习，生活在如今的时代，我们真的是很幸运。学习可以让我们学习到新的知识，明白更多的道理，提高自己的心理成熟度，完善自己的人格，提升自己的自我境界，体会人生的快乐。 <br><br>
  
         - 3. *更多人生选择*——记得短暂的大学期间经常看一些成功人士的故事，不过历经几年社会的“摧残”后，我发现现在很少有小学、初中毕业当老板的故事了，大部分都是知名学府毕业生，比如马云、马化腾、雷军、李彦宏等，这都是高等学府出来的精英人才，也诠释了学习是如何影响我们的人生选择的。学习到的知识就是金钱，不仅可以让我们获得更好的薪资待遇，还可以让我们有更好的发展前途，更高的成功几率。 龙应台写给儿子的话：孩子，我要求你读书用功，不是因为我要你跟别人比成绩， 而是因为，我希望你将来会拥有选择的权利，选择有意义、有时间的工作， 而不是被迫谋生。 当你的工作在你心中有意义， 你就有成就感。 当你的工作给你时间， 不剥夺你的生活， 你就有尊严。 成就感和尊严，给你快乐。<br><br>
 

    - 2. 学习的重要性？<br><br>
 
        - 1. *现实的压迫*<br><br>

          - 《北京爱情故事》中石小猛说过，“没钱没房子在这个城市里边你永远是外地人，你穿得跟墙壁一个色，你站在大街上一言不发，一动不动，永远有人找你查暂住证，稳定这两个字永远不属于你”。人生是一场旅行，可以经历许多你绝对意想不到的事情，甚至有时候你正满足于平稳的生活，却突然被下一刻的惊涛骇浪而受惊。这就是人生，充满了不确定性，又充满了必然性，正好诠释了什么是“世界上唯一不变的是变化：这句名言。<br><br>

          - 前几年出现了新词：房奴，大家一定很熟悉吧？每个打工仔都希望通过自己在异地的努力买到房，开上车，娶到老婆，可惜事与愿违，我们总是存不够买房的钱，买不起便宜的轿车，娶不到贤惠的妻子。这就是充满不确定性的现实，压得你喘不过气来，却又不得不踽踽前行。现实的压迫让我们不得不努力，不得不通过学习来学会一技之长，让自己在这个社会上有一个可以养家糊口的基本技能，不至于活得太辛苦。<br><br>

        - 2. *梦想的愿景*<br><br>

          - 初中学习陶渊明那句“采菊东篱下，悠然见南山”的时候你是否羡慕那种悠闲舒适的生活意境？学习刘禹锡那句“谈笑有鸿儒，往来无白丁“的时候你是否羡慕那种高朋满座的生活状态？说实话，我很羡慕，甚至那时候就幻想我如果有这种生活意境或者状态，我会是一个什么样的人，我有什么样的成就。想的更深时，就想怎么样才能做到人家那种生活状态。<br><br>

          - 这就是梦想的愿景，我们有了梦想，就要幻想自己拥有这种实现梦想的愿景，并且找到实现这种愿景的方法，还要去努力奋斗。那为什么要学习呢？肯定是我们想提高自己的人生修养，获得更丰富更深奥的知识，培养自己越加娴熟高深的金技能，锻炼我们的逻辑思维能力，丰富自己的生活和社会的各方面认识，学会更好的生活。<br><br>
 
          - 只有学习，才能让我们不断成长，当你体会到自己不断成长的时候，你就会发现，那种发自内心的快乐是其他物质无法带给我们的。只有学习，才能获得新知，增长才干，才能实现我们的梦想。<br><br>
 
       - 3. *自我的成长*<br><br>
         - 学习可以让我们明理，即明白为人处世的道理，掌握处理事情的方法，解决问题更加快捷迅速方便；学习可以让我们强大，人类发展到今天是一代又一代人不断努力的结果，我们通过学习可以掌握前人留下的宝贵财富，才能在社会上更好的立足。<br><br>
         - 周总理年少时就立下了宏大的志愿，为中华崛起而读书，结合当时时代状况来看，周总理的这个学习目的可谓是真正结合时代，体恤民众疾苦。<br><br>
         - 如果不学习，我们看待事物就可能比较片面，在解决事情的时候可能过早的下结论，错失良机。所以坚持学习、终身学习才是我们正确的成长道路，只有学习，才能让我们变得更有远见和富有意义。<br><br>
